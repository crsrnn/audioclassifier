Automated Audio Classifier Recycle Bin

Details

Welcome to the AI Audio Classifier Recycle Bin Hackaday project page. All the parts necessary for this build are detailed in the "Components" section. This project is made to be reproducible, by following the "Instructions" section you should be able to build one for yourself. The Code, Gerber files, STL files, editable, PCB schematic and design are all provided in the "Files" section. All the resources in this project are fully open source so you are free to use it or modify it. I've also added some tips, hardware upgrade options, testing videos, and a case study discussion in the "Logs" section so you can learn more about this project.

Illustrations

Uncovering the Unseen

Diagram 1

Electronic Connections System Diagram

The diagram illustrates a system designed for audio classification and motor control. Key components include the Arduino Pro Portenta C33 microcontroller, an AI Audio Classifier, a motor, a power supply, a Hall Effect sensor, and various other electronic components. The system operates by capturing audio input, processing it through the AI classifier, generating classification results, and sending control signals to the motor driver. The Hall Effect sensor provides feedback on the motor's position or speed, enabling precise control.

Diagram 2

IoT Cloud System Diagram

The diagram illustrates a smart waste management system for urban parks. It involves IoT nodes equipped with microcontrollers and AI audio classifiers for waste classification. These nodes transmit data to the cloud via WiFi and gateway routers. The cloud processes the data and updates a web-based dashboard for monitoring and control. This system enables efficient waste management and resource optimization.

Diagram 3

Electronic Connections System Diagram (Old version, base unit)

The diagram illustrates a system for controlling a stepper motor and a servo motor using an Arduino Nano 33 BLE Sense microcontroller. Key components include a stepper driver, a hall effect sensor, a power supply, and the motors themselves. The microcontroller generates control signals, the stepper driver amplifies them, and the hall effect sensor provides position feedback. The servo motor receives direct control signals from the microcontroller.
